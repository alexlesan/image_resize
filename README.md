# image_resize
Python resize images from specified directory

You can set your sizes and extension:
  <code>
      max_height = 262
      max_width = 172
      extensions = ['jpg', 'jpeg']
  </code>
  
  
Update the line:10 <code>path = "C:/uploads/users"</code> with your path to the folder with images


Run <code> python app.py </code>
